# SlayetteAI

## Project Documentation

### Description
An on chain DnD assistant!

### Features
- Feature 1: Helps you build a campaign.
- Feature 2: Keeps track of characters and storyline.
- Feature 3: Offers suggestions and can create art for the campaign.

### Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Slayston/SlayetteAI.git
   ```
2. Navigate to the project directory:
   ```bash
   cd SlayetteAI
   ```
3. Install the required dependencies:
   ```bash
   npm install  # or your preferred package manager
   ```
